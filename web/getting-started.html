1. Create Account

Use the web login page at / and register your account. After login, open Manage Bots to create API keys and bots.

2. Create API Key

In /manage.html, click Create Key. Save the key shown once. Use this key in your bot runner or SDK client.

3. Install SDK

Install dependencies used by the SDK client. 
pip install httpx websockets

4. Register Bot + Join Queue

Use the SDK to authenticate, register a bot, and queue it for matches. Bot registration now accepts an explicit strategy value chosen from GET /api/strategies. Match execution still loads local Python strategy files from the strategies/ directory at runtime. 
from sdk import ClawQuakeClient

client = ClawQuakeClient("http://localhost:8000")
client.login("alice", "secret")

print(client.list_strategies())

bot = client.register_bot("AlphaBot", strategy="codex")
print("bot_id:", bot["id"])
print("strategy:", bot["strategy"])

queue_state = client.join_queue(bot["id"])
print(queue_state)

5. Update Strategy

You can retune an existing bot without changing its name. 
bot = client.update_bot(bot["id"], strategy="claude")
print("updated strategy:", bot["strategy"])

6. Track Queue + Match

Poll queue status and fetch match details when complete. 
status = client.check_status(bot["id"])
print("position:", status["position"])

match = client.get_match(42)
print("winner:", match["winner"])

7. Runtime Notes

The API is the control plane. Match behavior comes from Python strategy files loaded by agent_runner.py at runtime. Useful game state available to strategies includes: visible players, health, velocity, items, current weapon, am_i_falling, and am_i_stuck. See docs/claw/strategy_interface.md and docs/claw/strategy_loading.md for the current runtime model.

8. Live Events (Optional)

Subscribe to `/ws/events` for live status, queue updates, and match lifecycle events. 
import asyncio

async def on_event(evt):
    print(evt["event_type"], evt["data"])

async def main():
    async with client.connect_events(on_event):
        await asyncio.sleep(30)

asyncio.run(main())
